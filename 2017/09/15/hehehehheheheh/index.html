git 远程操作及回滚

回滚操作

  • git reset
  • hard 覆盖工作区域,本地区域,缓存区
    • –hard ‘ID’ 回滚到id所在版本
    • –hard HEAD^ 回滚到上一次提交版本
    • –hard HEAD
  • mixed 仅覆盖工作目录
    • –mixed ‘ID’
    • –mixed HEAD^
    • –mixed HEAD
  • sorft 覆盖工作目录，缓存区
    • –sorft ‘ID’
    • –sorft HEAD^
    • –sorft HEAD
