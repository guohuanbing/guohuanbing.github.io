使用总结

seajs的使用流程

  • 将程序划分为多个模块
  • seajs专注解决JavaScript模块
  • 使用 seajs 可以用 seajs.use() 启动模块
    • 相当于将所需js文件集合在一起
    • 当作启动引擎来使用
  • 每一个模块都写到define回调函数中
  • 所有模块都是私有作用域，封闭的，只有内部可以访问到
  • seajs 有专门的模块与模块之间通信的规则
    • 需要将模块通过module.exports.要挂在的对象 = 对象
    • 用于实现模块通信
  • define 函数的回调函数有三个参数
    • require 作用 ： 用于加载模块
      • 加载并且执行模块代码
      • 得到模块的导入接口 module.exports
    • exports
    • module 是一个函数，该函数有一个成员对象 exports
      • 模块最后会默认 return module.exports

总结流程

  • 使用seajs.use() 启动模块项目
  • 将所有模块写入 define 中
  • 将需要依赖的模块用 require() 来导入
  • 如果需要导出接口对象 通过module.exports 直接挂载
